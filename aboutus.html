About Us

Our Story

Sculpture Management System is a web-based application that allows users to search for sculptures from temples located in different states and cities. The user can search for a particular sculpture by entering the name of the sculpture or the name of the temple. The application provides detailed information about the sculpture, including its history, material used, and other relevant details.The system also includes an admin panel that enables the admin to manage the temples, sculptures, and other related data. The admin can add new temples to the system and manage the sculptures associated with each temple. The admin can also edit and delete the existing temples and sculptures.
